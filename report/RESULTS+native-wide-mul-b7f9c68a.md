# Benchmark results

128-bit integer division: a `u64`-limb reciprocal path vs the native
`u128` / `i128` operators (`__udivti3` / `__umodti3`), across native, Node
(V8) and wasmtime (Cranelift). Generated by `scripts/report.py` from the
saved criterion runs.

## Features

- [x] native wide multiply

- **CPU**: Intel(R) Core(TM) i9-9880H CPU @ 2.30GHz
- **OS**: macOS 15.7.7 (x86_64)
- **Cores**: 16
- **rustc**: rustc 1.98.0-nightly (61d7280f3 2026-06-06)
- **Node**: v26.0.0
- **wasmtime**: wasmtime 45.0.1 (83166ba31 2026-06-05)

# Benchmark results

128-bit integer division: a reciprocal path (native u128 arithmetic that
lowers to the wide-arithmetic proposal) vs the `u128` / `i128` operators
(`__udivti3` / `__umodti3`), across native, V8 (d8) and wasmtime (Cranelift).
Generated by `scripts/report.py` from the saved criterion runs.
